# Adult Income Prediction

## About

This project uses machine learning models to predict whether an adult earns more than $50K per year using demographic and employment-related features.

The project compares:
- Logistic Regression
- Random Forest

using the Adult Income dataset.

## Features

- Data preprocessing
- One-hot encoding
- Model training
- Hyperparameter tuning
- Classification analysis
- Model comparison
- Performance evaluation

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Models Used

### Logistic Regression
- Binary classification
- Hyperparameter tuning with GridSearchCV
- Accuracy: ~85.5%

### Random Forest
- Ensemble learning
- Hyperparameter tuning
- Accuracy: ~86.8%

## Findings

Random Forest achieved higher accuracy and better captured non-linear relationships in the dataset compared to Logistic Regression.
